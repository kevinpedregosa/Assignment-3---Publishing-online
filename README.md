# Assignment-3---Publishing-online

ICS 32 Assignment 3 – Journal + DSP Client
Kevin Noel Pedregosa

This program builds on the Assignment 2 journaling application by adding the ability to connect to the ICS 32 Distributed Social Platform (DSP) server through network sockets. Users can create and manage journal entries locally in .dsu files and also publish posts and profile bios to the DSP server. The program includes both a command-line admin mode and a user-friendly menu interface. It communicates with the DSP server through the DSP JSON protocol to authenticate users, publish posts, and update profile bios.
